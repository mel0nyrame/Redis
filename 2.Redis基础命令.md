# Redis基础命令

## 基础命令

key-value数据结构,key为String,value有多种多样的形式

| value数据类型 | 示例 |
| :---: | :---: |
| `String` | hello world |
| `Hash` | {name: "Jack", age: 21} |
| `List` | [A -> B -> C -> C] |
| `Set` | {A, B, C} |
| `SortedSet` | {A: 1, B: 2, C: 3} |
| `GEO` | {A: (120.3, 30.5)} |
| `BitMap` | 0110110101110101011 |
| `HyperLog` | 0110110101110101011 |

### 通用命令

命令均可在[redis官网](https://redis.io/docs/latest/commands)查询

- 命令
  - KEYS [pattern]:根据规则匹配相对应的key
  - DEL [key...]:批量删除key
  - EXIST [key..]:判断key是否存在
  - EXPIRE [key second]:设置key的存活时间
  - TTL [key]:查看key的存活时间
